# Students Marks Analyzer App

A simple Streamlit app to calculate total marks, percentage, and grade for a student.

## Files

- `app.py` - Streamlit application source
- `requirements.txt` - Python dependencies

## Run locally

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Start the app:
   ```bash
   streamlit run app.py
   ```
3. Open the browser at `http://localhost:8501`

## Deployment with Streamlit Cloud

1. Go to [Streamlit Cloud](https://streamlit.io/cloud).
2. Connect your GitHub account.
3. Select the repository `Sejal-PS/Students-Marks-Analyzer-App`.
4. Set the main file to `app.py`.
5. Deploy the app.

## Notes

- The app uses `streamlit` only.
- You can update the browser title and layout in `app.py` as needed.
